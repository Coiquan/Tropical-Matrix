所以修改了排序规则，在这个定义下就没问题了。  
发现问题总归是个好消息。但之前的数据全都作废了挺可惜的。
下一步想这么改：  
想把程序改成从n=2开始，对每一个n分别筛选。因为目前一次循环只能处理一个文件，所以文件一多筛选效率会大打折扣，出结果很慢。
